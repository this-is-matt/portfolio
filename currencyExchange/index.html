Exchange Rate Apps using ExchangeRate-API

Welcome to the currency exchange center. This project is actively in development. The below sections will discuss the status of each portion of the project.

Conversion Pair App

The Conversion Pair app is quite useful. This application uses an api call that takes the base currency code, the target currency code, and amount you wish to convert. It then simply display the conversion for that amount.
There are some really cool things going on in this application. The first great thing is that the select form inputs are dynamically populated from a JSON file that contains all of the currency code information.
This way we ensure there are no errors that are caused by typos or user error in other ways. This application will default to one US Dollar as the base Currency and amount, this is the standard conversion rate that we often here about.
I would still like to add some higher quality client side validation. Another thing that i want to add is a better formatted output that includes more information about the currencies and their countries. I would also add a section that would tell when the conversion rates were last updated and when they will be updated next.

Full Conversion Chart App

The Full Conversion Chart App is still very much under development. In fact, it is currently non operable. But when it is finished, the user will input a single base Currency Code and they will receive back a well formatted table that includes the conversion rates for every country. This is not incredibly useful, more just informational.
For this application to be completed, i need to fix the fetch that gathers the data and adjust the processing functions. They are not working properly.
